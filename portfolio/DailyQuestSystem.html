ABOUT

The DAILY QUEST SYSTEM is a UI recreation of popular anime/manhwa Solo Leveling's system. In this series, the system plays a massive role both to the main character and narrative. As a UI development project, I decided to recreate the comics system and make it functional.


Recreating Solo Leveling's System

The system windows appear towards the start of the series and continue well into it. Throughout the story, they retian the same style. However, they're always angled, covered in speech bubbles, or have some other transformation upon them.

Part of the challenge of this project was accurately recreating these windows, which included tracking down the original font, matching the font-decoration, recreating the border, and adding the different menus.

Creating a Low-Fi Prototype

Before creating the full version, I went ahead and made a quick low-fi prototype with the objective of identifying all necessary features. The low-fi prototype was created in Fluid UI and can be accessed online. Try it here!
